# 🌱 The Garden of Unanswered Questions

An immersive, interactive web experience where questions become a living world.

Instead of presenting answers, this project explores curiosity as an evolving digital garden — where every interaction shapes the environment.

---

## 🌌 Concept

The website is built around a simple idea:

> What if questions could grow into worlds?

Each question becomes a living element that reacts to the user, creating a dynamic and emotional experience.

---

## 🌿 Core Elements

- ☁️ Cloud of Overthinking — floating thoughts and doubts  
- 🌊 River of Possibilities — flowing future paths  
- 🌫️ Valley of Almost — unfinished ideas and choices  
- 🌱 Blooming Questions — interactive curiosity nodes  
- 🌳 Tree of Unfinished Answers — grows with every interaction  

---

## ⚡ Features

- Interactive clickable questions  
- Dynamic particle background  
- Ripple click effects  
- Growing visual tree system  
- Ambient glowing UI atmosphere  
- Fully single-file implementation (HTML + CSS + JS)

---

## 🎨 Experience

This is not a static website.

It is a reactive environment that responds to curiosity.

> Click. Explore. Grow.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Glassmorphism + Gradients)  
- Vanilla JavaScript  
- Canvas API (Particles animation)

---

## 🚀 How to Run

1. Download or clone the repository  
2. Open `index.html` in any modern browser  
3. Click the core to activate the garden  

No installation required.
